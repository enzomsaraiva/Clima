# 🌤️ Clima

Um site simples, moderno e responsivo para verificar as condições climáticas atuais de diversas regiões em tempo real.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias fundamentais da web:
* **HTML5**: Estrutura semântica da página.
* **CSS3**: Estilização moderna e layout responsivo.
* **JavaScript (ES6+)**: Consumo de API de clima e manipulação dinâmica do DOM.

## 📦 Estrutura do Projeto

O repositório está organizado da seguinte forma:
```text
├── assets/        # Imagens e fundos dinâmicos
├── css/           # Arquivos de estilização (Estilos globais e responsividade)
├── js/            # Scripts e lógica de integração com a API
└── index.html     # Página principal da aplicação
```

## ✨ Funcionalidades Planejadas / Atuais

* [ ] Busca de clima por nome da cidade.
* [ ] Exibição de temperatura, umidade e velocidade do vento.
* [ ] Mudança dinâmica de fundo com base no clima atual (sol, chuva, neve).
* [ ] Design totalmente responsivo para celulares, tablets e computadores.

## 🛠️ Como Executar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com
   ```
2. **Navegue até a pasta:**
   ```bash
   cd Clima
   ```
3. **Abra o projeto:**
   * Basta dar um duplo clique no arquivo `index.html` ou utilizar a extensão **Live Server** no VS Code.
